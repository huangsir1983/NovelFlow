# 涛割 (Taoge) - AI视频生成平台 开发进度报告

**报告日期**: 2026-02-08
**版本**: 1.1.0

---

## 项目概述

基于 PyQt6 的桌面应用，核心功能是 AI短剧视频生成，采用"先图后视频(I2V)"工作流，支持多模型智能路由。

|------|------|

| 源代码文件 | 59 个 (src/) |

| 总代码行数 | ~20,000+ 行 |
| 测试代码行数 | ~2,414 行 |
| 数据库大小 | 28KB (SQLite) |

## 已完成模块

### 1. 数据库层 (database/)
- 8 个 ORM 数据模型: Project, Scene, Character, Task, CostRecord, CostSummary, **Prop, SceneProp**
- SQLAlchemy + SQLite
- 完整的关系映射和会话管理
- **新增 Prop(道具)模型**：支持道具名称、类型、参考图、AI描述、全局/项目维度
- **新增 SceneProp(场景-道具关联)模型**：支持道具在场景中的位置和缩放

### 2. 生成服务层 (services/generation/)
- BaseProvider 基类 (Strategy模式)
- 4 个闭源 API Provider: Vidu、Kling、即梦
- 1 个开源 Provider: ComfyUI
- ModelRouter 智能模型路由器 (支持6种场景类型 + 4种质量等级)
- ImageGenService 图像生成服务

### 3. 场景处理 (services/scene/)
- SceneProcessor: SRT字幕解析、场景分组 (duration/content两种策略)
- PromptGenerator: 支持6种风格、8种情绪的Prompt生成
- ImageFirstWorkflow: 先图后视频工作流、首尾帧精确控制

### 4. 任务队列 (services/task_queue/)
- 优先级队列 (URGENT/HIGH/NORMAL/LOW/BACKGROUND)
- 异步任务执行、重试机制
- 进度通知回调、持久化支持

### 5. 成本追踪 (services/cost/)
- 实时积分余额查询
- 积分消耗记录和预警机制
- 成本报表生成

### 6. 导出服务 (services/export/)
- 剪映项目导出 (图像轨道、视频轨道、字幕轨道)
- 字幕字体/大小/位置自定义

### 7. AI 分析服务 (services/ai_analyzer.py)
- **AI 画面提示词生成** (TYPE_IMAGE_PROMPT)：一键分析文案生成中文画面描述
- **AI 视频提示词生成** (TYPE_VIDEO_PROMPT)：综合分析运镜方式、景别、角色动作、表情变化、特效
- **一键全量分析** (TYPE_ALL)：同时生成画面提示词 + 视频提示词 + 5维子分析
- **AI 角色生成** (TYPE_CHARACTER_GENERATE)：从文案自动提取角色描述
- **AI 道具生成** (TYPE_PROP_GENERATE)：从文案自动提取道具列表
- 所有 AI 输出改为中文，Prompt 模板全面重构
- 基于 QThread 的异步执行，带进度信号和错误回调

### 8. 控制器层 (services/controllers/)
- ProjectController: 项目控制器
- GenerationController: 生成控制器
- MaterialController: 素材控制器
- CanvasController: 画布控制器
- **PropController: 道具控制器** (新增 — 道具 CRUD + 场景道具关联管理)

### 9. UI层 (ui/)
- MainWindow 主窗口 (双模式: 向导+画布)
- 16 个 UI 组件:
  - ProjectsPage - 项目管理
  - SceneEditorPage - 场景编辑 (集成画布/编辑双模式切换、道具管理)
  - SceneCard - 场景卡片
  - ShotPropertyPanel - 镜头属性面板 (**重构** — 新增视频提示词区块，移除运镜下拉框)
  - VideoPreviewPanel - 视频预览
  - FirstLastFrame - 首尾帧控制
  - CanvasMode - 画布模式 (**增强** — 拖放接收、属性转发、侧边栏集成)
  - **CanvasSidebar - 画布左侧边栏** (新增 — 角色区+道具区，支持拖拽到场景卡片)
  - **CanvasPropertyPanel - 画布属性面板** (重构 — 视频提示词区块、AI按钮重命名)
  - MaterialsPage - 素材库
  - TasksPage - 任务管理
  - SettingsPage - 设置页面 (937行)
  - SrtImportDialog - SRT导入对话框
  - ScriptStructurePanel - 脚本结构面板
  - StoryboardAnalysisDialog - 分镜分析对话框
  - PixmapCache - 图片缓存
- 深色主题 (Dark Theme QSS)
- 最小窗口尺寸: 1400x900

### 10. 测试 (tests/)
- 10 个测试文件, 2,414 行测试代码
- 覆盖: 配置、模型、场景处理、Prompt生成、模型路由、任务队列、服务层、集成测试
- 框架: pytest + pytest-asyncio

### 11. 配置 (config/)
- settings.py: JSON配置管理
- constants.py: Prompt模板、动作库等常量 (**重构** — 全部改为中文模板，新增视频提示词/角色/道具生成模板)

## 技术架构

- **设计模式**: Strategy + Factory + Router
- **前端**: PyQt6 桌面GUI
- **后端**: Python 3.x + asyncio + aiohttp
- **数据库**: SQLite + SQLAlchemy ORM
- **测试**: pytest + pytest-asyncio
