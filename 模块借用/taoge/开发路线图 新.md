## 一、项目总览

基于 PyQt6 的桌面应用，核心功能是 AI 短剧视频生成，采用"先图后视频(I2V) + 多模型智能路由 + 标准化精品流程"模式。融合参考软件后，工作流形成完整闭环：脚本输入 → 分镜/故事板生成 → 图片/视频生成 → 导出。强调角色一致性、风格自定义、快捷优化，以及电商/慢剧扩展。

| 指标 | 数值 |
|------|------|
| Python 文件数 | ~85 个 |
| 源代码文件 | ~70 个 (src/) |
| 测试文件 | 15 个 (tests/) |
| 总代码行数 | ~35,000+ 行 |
| 数据库 | SQLite + SQLAlchemy ORM |
| 五层架构 | UI → Controller → Service → Data → External |

## 五、开发优先级

1. **高优先级 (P1)**: 故事板集成（StoryboardService/Controller），分镜交互（画布拖拽/排序），视频路由扩展（GenerationService）。
2. **中优先级 (P2)**: 快捷功能（QuickFuncController），风格库（StyleService），错误重试（TaskQueue）。
3. **低优先级 (P3)**: 电商扩展，回收站，UI优化（暗黑模式）。

**里程碑**:
- v1.2.0: 融合核心闭环（故事板 → 生成 → 导出）
- v1.3.0: 完整测试 + 优化
- v2.0.0: 电商/慢剧模板发布
