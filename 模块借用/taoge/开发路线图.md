# 涛割 (Taoge) - AI视频生成平台 开发路线图

**更新日期**: 2026-02-10
**当前版本**: 1.1.0
**项目状态**: 中后期开发阶段

---

## 一、项目总览

基于 PyQt6 的桌面应用，核心功能是 AI 短剧视频生成，采用"先图后视频(I2V)"工作流，支持多模型智能路由。

| 指标 | 数值 |
|------|------|
| Python 文件数 | ~70 个 |
| 源代码文件 | ~60 个 (src/) |
| 测试文件 | 10 个 (tests/) |
| 总代码行数 | ~28,000+ 行 |
| 数据库 | SQLite + SQLAlchemy ORM |
| 五层架构 | UI → Controller → Service → Data → External |

## 六、技术架构

```
┌──────────────────────────────────────────┐
│  UI Layer (PyQt6)                        │
│  30个组件 · 四区域画布 · 深色/浅色主题     │
│  MainWindow → InfiniteCanvasPage         │
│    ├── ScriptZone (剧本区)               │
│    ├── CharacterPropZone (角色道具区)     │
│    ├── DirectorZone (导演画布区)          │
│    │     ├── DirectorCanvasPage (总览)    │
│    │     └── SceneWorkCanvas (场景画布)   │
│    └── PreEditZone (预编辑区)            │
└──────────────────────────────────────────┘
                    ↓ 信号 (pyqtSignal)
┌──────────────────────────────────────────┐
│  Controller Layer                        │
│  ProjectController · GenerationController│
│  CanvasController · MaterialController   │
│  PropController                          │
└──────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────┐
│  Service Layer                           │
│  Generation · Scene · TaskQueue · Cost   │
│  AIAnalyzer · Export                     │
└──────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────┐
│  Data Layer (SQLAlchemy ORM)             │
│  8个模型 · SQLite · 会话管理             │
└──────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────┐
│  External Integration Layer              │
│  Vidu · Kling · Jimeng · ComfyUI         │
│  DeepSeek LLM · 剪映导出                 │
└──────────────────────────────────────────┘
```

**设计模式**: Strategy · Factory · Router · Observer · Singleton · Context Manager
