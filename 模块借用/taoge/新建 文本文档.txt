Context

 涛割(Taoge)是一个基于PyQt6的AI短剧视频生成平台，当前处于中后期开发阶段。画布模式(canvas_mode.py,
 525行)已有基础功能（场景卡片、连线、三种排列、缩放），但缺乏深度交互能力。用户要求实现两阶段工作流：

 1. 第一阶段 - 分镜分析：导入文案后，先提取角色、分场景、调整分镜
 2. 第二阶段 - 无限画布：确认分镜后进入无限画布，关联角色到场景，点击卡片右侧弹出属性面板，支持右键菜单、批量生成、生成进度可视化等

 第一阶段 - 分镜分析

 流程

 导入文案 → 步骤1:角色提取 → 步骤2:场景分割 → 步骤3:分镜调整 → 确认写入DB → 进入画布

 StoryboardAnalysisDialog (三步骤向导对话框)

 - 使用 QStackedWidget 管理三个步骤页面
 - 顶部 StepIndicatorBar 显示进度（圆点+连线样式）
 - 上一步/下一步/确认按钮
 - 信号: analysis_completed(int, list, list) → project_id, scenes, characters

 CharacterExtractionWidget (步骤1)

 - "AI提取角色"按钮 → QThread中分析文本提取角色
 - 角色卡片列表：名称、类型下拉、外貌描述、参考图上传、删除
 - "手动添加角色"按钮
 - 回退方案：正则匹配中文人名 + 从ai_tags["角色"]去重

 SceneAdjustmentWidget (步骤3)

 - 场景预览卡片列表，支持多选（Ctrl+Click）
 - 工具栏：合并选中、拆分选中、上移、下移
 - 复用 SceneProcessor 的分组逻辑

 第二阶段 - 无限画布增强

 SceneCanvasCard 增强 (canvas_mode.py)

 - 新增属性：_characters, _generation_progress, _is_generating, _group_id
 - 新增方法：set_characters(), set_generation_progress(), refresh_thumbnail(), update_scene_data()

 - 新增信号：card_double_clicked(int), cards_multi_selected(list), card_context_menu_requested(int, QPointF)

 - 分组矩形：add_group_rect() 为章节卡片绘制半透明包围框
 - 卡片数据更新API：update_card(), update_card_progress(), update_card_characters()

 - 工具栏新增：批量生成按钮、分组管理按钮、缩放百分比显示

 - 右键菜单 → 委托给 canvas_context_menu.show_canvas_context_menu()
 - 对外API：update_scene_data(), update_generation_progress()

 CanvasPropertyPanel (新文件)

 - 精简版 ShotPropertyPanel，适配画布模式浮动/可关闭特性
 - 复用 CollapsibleSection 和 PropertyRow 组件
 - 布局：关闭按钮+场景序号 | 大图预览 | 叙事内容 | 视觉设定 | 运镜 | 角色 | 生成按钮
 - 信号：property_changed, generate_image_requested, generate_video_requested, close_requested

 canvas_context_menu (新文件)

 - 菜单项：生成图片/视频、打开属性面板、批量生成(多选时)、复制/删除场景、设为章节起始

 CanvasController (新文件)

 - 场景CRUD：delete_scene, duplicate_scene

 技术注意事项

 1. 线程安全：AI角色提取在QThread中执行，结果通过信号传回主线程
 2. 数据一致性：画布和编辑器共享 SceneEditorPage.scenes_data 作单一数据源
 3. 内存管理：使用 PixmapCache LRU 策略，生成后 invalidate() 再 refresh_thumbnail()
 4. 向后兼容：现有类以追加方法方式增强，原有信号和接口保持不变
 5. 性能：超100个场景时考虑切换 SmartViewportUpdate
 6. 数据库：Project.source_content 字段已有，无需新增表结构

 验证方式

 1. 导入一个SRT文件 → 触发分镜分析对话框 → 完成三步骤 → 进入画布
 2. 画布中单击/双击卡片 → 右侧属性面板弹出 → 编辑属性 → 数据库已保存
 3. 右键卡片 → 菜单操作（生成图片、复制、删除） → 卡片状态正确更新
 4. Ctrl多选 → 批量生成 → 所有选中卡片显示进度覆盖层
 5. 拖拽卡片重排 → 切回编辑器模式 → 场景顺序同步
 6. 网格/时间线/故事板三种排列仍正常工作
