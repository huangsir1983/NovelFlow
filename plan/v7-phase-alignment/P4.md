## 计划

### 1. 本阶段定位

P4 重点不是再扩页面，而是把“专业判断系统”接进三段闭环，特别是 Cinema Lab、导演方法论和多 Agent 协同。

### 2. 交付范围

- Cinema Lab 六模块
- 导演 7 步法
- 声音叙事
- 场面调度
- 视觉母题
- 跨文化预设
- 6 个在线协同 Agent
- 知识引擎三段被动建议
- Board 双层视图：
  - 创作视图
  - 流程视图

### 3. 阶段结果定义

Hidden 用户应获得“专业级判断增强”，而不只是更多按钮。复杂能力要通过三段各自的上下文自然进入。

## 验证

### 1. 验收标准

- Cinema Lab 六模块可实际驱动创作
- 导演 7 步不是静态说明，而是可推进的工作流
- 6 Agent 稳定在线协同
- 知识引擎在三段都能提供有上下文的被动建议

### 2. 退出门槛

- `Hidden` 能覆盖专业导演和方案层核心使用场景
- 流程视图只在需要时出现，不破坏默认创作视图

## 修复闭环

### 1. 高频问题池

- Agent 互相打架
- Prompt 漂移导致输出不稳定
- Cinema Lab 参数无法正确反映到结果
- 知识建议与当前阶段不相关

### 2. 修复策略

1. 为每类问题沉淀固定测试样片
2. 把失败样例写入回归集
3. 区分规则问题、Prompt 问题、编排问题
4. 重跑多 Agent 集成测试
5. 记录成本与稳定性变化

### 3. 不允许带入 P5 的问题

- 6 Agent 长期不稳定
- Cinema Lab 输出不可依赖
- Hidden 能力需要频繁回退人工兜底
