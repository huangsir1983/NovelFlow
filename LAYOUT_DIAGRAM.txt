DYNAMIC CALCULATION FLOW
═══════════════════════════════════════════════════════════════════════════════════════════════════

    buildCanvasGraph(scenes, shots, options)
           │
           ├─ Sort scenes by order
           │
           ├─ Initialize: currentY = 0
           │
           └─► For each scene:
               │
               ├─ baseY = currentY
               │
               ├─ Calculate shotsGroupHeight = 
               │   Math.max(0, sceneShots.length - 1) * SHOT_GAP
               │
               ├─ Create Scene Node at:
               │   x: COL_X[0] = 0
               │   y: baseY + shotsGroupHeight / 2
               │
               └─► For each shot in scene:
                   │
                   ├─ shotY = baseY + shotIndex * SHOT_GAP
                   │
                   ├─ Create Shot Node at:
                   │   x: COL_X[1] = 420
                   │   y: shotY
                   │
                   ├─ Create Prompt Node at:
                   │   x: COL_X[2] = 860
                   │   y: shotY
                   │
                   ├─ Create Image Node at:
                   │   x: COL_X[3] = 1300
                   │   y: shotY
                   │
                   └─ Create Video Node at:
                       x: COL_X[4] = 1740
                       y: shotY
               
               After all shots:
               shotsHeight = Math.max(1, sceneShots.length) * SHOT_GAP
               currentY += Math.max(NODE_HEIGHT, shotsHeight) + SCENE_PADDING
